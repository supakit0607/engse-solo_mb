🗑️ Community Waste Tracker
ระบบจัดการและแจ้งจุดทิ้งขยะในชุมชน

📌 รายละเอียดโครงการ
Community Waste Tracker เป็นแอปพลิเคชันสำหรับบันทึกและตรวจสอบจุดทิ้งขยะประเภทต่าง ๆ ในชุมชน เช่น ขยะรีไซเคิล ขยะทั่วไป หรือขยะอันตราย เพื่อให้คนในพื้นที่สามารถจัดการขยะได้อย่างถูกวิธีและรักษาสิ่งแวดล้อมในชุมชนให้ยั่งยืน

แอปพลิเคชันนี้พัฒนาด้วย Flutter และใช้ SQLite เป็นฐานข้อมูลภายในเครื่อง รวมถึงใช้ Provider ในการจัดการ State ของแอปพลิเคชัน

🎯 จุดประสงค์ของแอป
เพื่อให้ประชาชนสามารถระบุตำแหน่งจุดทิ้งขยะแยกประเภทได้สะดวก

เพื่อช่วยให้การจัดการขยะในชุมชนมีประสิทธิภาพมากขึ้น

เพื่อเก็บรวบรวมข้อมูลจุดทิ้งขยะไว้เป็นฐานข้อมูลสำหรับการตรวจสอบในอนาคต

⚙️ เทคโนโลยีที่ใช้
Flutter: ใช้พัฒนาแอปพลิเคชันมือถือ

Provider: ใช้จัดการ State ของแอป

SQLite (sqflite): ใช้จัดเก็บข้อมูลขยะในเครื่อง

Dart: ภาษาโปรแกรมหลักในการพัฒนา

✨ ความสามารถของแอป (Features)

เลือกประเภทขยะแยกตามหมวดหมู่

บันทึกข้อมูลแบบ Offline ลงฐานข้อมูล SQLite
